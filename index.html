jQuery - scrollslider

This plugin allows you to slide, scroll, step through the children of any HTML element.
The clou: It wont touch your markup.

Let's start with the »markup or see »the index

Markup

The markup is quite simple, just use a default <ul> tag with its <li> children or any other structure of your choice with a parent and its children tags.

<ul>
    <li>item 1</li>
    <li>item 2</li>
    …
</ul>

After you've written your markup, the only thing you've to do is to start the »initialization of the plugin.

Initialization

To initialize the plugin just use this little snippet inside your javascript

$( 'ul' ).scrollslider();

and it's done.

You shouldn't concern about the user's browsers. It has a high »compatibility to many browsers.

Compatibility

This plugin is actually tested with FireFox 16, Crome 22, Opera 12, Safari 5.1 and IE 9.

It's also compatible with IE 8 and IE 7 by including this fix for them. For example with conditional comments.

<!--[if lt IE 9]>
    <script type="text/javascript" src="ie-fixes.js"></script>
<![endif]-->

Now your instance is running, you could take a look at the possible »options to customize the behaviour.

For detailed information, please see the project's readme.

To interact with your instance, you're able to do so with some integrated »methods.

Methods

At the moment you can see all available methods in the project's readme.

Interaction is also possible with »events

Events

The plugin provides some predefined events.

scrollslider-show, scrollslider-hide
Will be triggered to child elements, being visible (resp. invisible) when the current action completes. These events are called immediately before the action starts.
scrollslider-shown, scrollslider-hidden
Will be triggered to child elements, are visible (resp. invisible) when the latest action completes. These events are called after the slider stops/finished its scrolling.
